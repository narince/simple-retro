# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-30

### Added
- **Dynamic Board Templates**: Board creation now supports valid previews for Start/Stop/Continue, Mad/Sad/Glad, Lean Coffee, and more.
- **Immersive Reactions**: New center-stage animations for Applause, Rocket, Bulb, Star, and Gem.
- **Localization**: Full English and Turkish support.
- **Documentation**: Comprehensive README, CONTRIBUTING guidelines, and LICENSE.

### Changed
- **Reactions**: Refined visual effects to remove "falling rain" clutter, focusing on high-quality central animations.
- **UI/UX**: Improved board dialogs and overall responsiveness.

### Removed
- **Legacy Assets**: Removed unused `vozol` assets and debug scripts.
